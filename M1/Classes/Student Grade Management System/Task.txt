Assignment 1: Student Grade Management System (10 mins)
Requirements:
Create a Student class with the following:

Properties: Name, StudentID, Grades (List of integers)
Auto-implemented property for EnrollmentYear
Constructor that takes name and student ID
Method AddGrade(int grade) - validates grade is between 0-100
Method GetAverage() - returns average grade
Property Status (read-only) - returns "Pass" if average >= 50, else "Fail"
Static counter to track total students created

Task:

Get user input to create 3 students
Add 4 grades for each student
Display each student's details and status
Show total students created
